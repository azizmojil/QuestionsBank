Install the Design system libraries

This guide offers concise instructions for installing the Platform library into their workspace, ensuring a smooth setup and seamless incorporation of components and design language assets.

Installing and publishing the foundations and icons library files to design files.

To use the UI kit library files correctly, you must start by installing the foundation library and the icon library files first. These libraries contain atomic-level elements that form the base for the UI component library.

Open Figma.

Open Figma and log in to your account.

Import the “PC 1.0 Foundations” and “PC 1.0 icons” library files on your project folder.

Import the “PC 1.0 Foundations” and “PC 1.0 Icons” library files into your project folder. Ensure that you have created both a team file and a project folder within the team file.

Open the “PC 1.0 Foundations” and “PC 1.0 icons” library files on Figma.

Open the “PC 1.0 Foundations” and “PC 1.0 icons” library files, which will be used in another file (your Figma design file).

Publish the “PC 1.0 Foundations” and “PC 1.0 icons” library files on Figma.

Publish the “PC 1.0 Foundations” and “PC 1.0 icons” library files by clicking on the library icon at the top of the screen, then publish the library to the team that contains the design files.

Add the “PC 1.0 Foundations” and “PC 1.0 icons” library files to your design file.

To add the assets library files to the design file, use the library files icon at the top of the screen, then add the “PC 1.0 Components - Desktop” library by clicking on 'Add to file' button.

Installing and publishing the components library file to your design files.

Now, let's install our components library. It contains all the UI elements you need to create an exceptional digital experience for your products.

Import the PC 1.0 Components - Desktop” or “PC 1.0 Components - Mobile” on your project folder.

Import the “PC 1.0 Components - Desktop” or “PC 1.0 Components - Mobile” library files into your project folder. Ensure that you have created both a team file and a project folder within the team file.

Open the “PC 1.0 Components - Desktop” or “PC 1.0 Components - Mobile” library file on Figma.

Open the “PC 1.0 Components - Desktop” or “PC 1.0 Components - Mobile” library file, which will be used in another file (your Figma design file).

Publish the “PC 1.0 Components - Desktop” or “PC 1.0 Components - Mobile” library file.

Publish the “PC 1.0 Components - Desktop” or “PC 1.0 Components - Mobile” library file by clicking on the Library icon at the top of the screen, then publish the library to the team that contains your design files.

Add the “PC 1.0 Components - Desktop” Library file to your design file.

To add the Component library file to the design file, click on the Library icon at the top of the screen, then add the “PC 1.0 Components - Desktop” library by clicking on “Add to file” button.
